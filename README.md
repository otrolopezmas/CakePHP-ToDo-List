# CakePHP-ToDo-List

CakePHP version 2.8

Features:
- You can import your own todo list from Abstract Spoon ToDo software

List fields:
- Checkbox: allows to complete a task.
- Title
- Color (title)
- Priority (1 to 5)
- Percentage done
- Due date
- Status
- Categories (Multiple)
- Tags (Multiple)
- Comment (Long comment)
- Link (Macros)
